# SCOUTS

An interactive narrative game based on the play SCOUTS by Ryann Murphy.

SCOUTS was read at Juilliard, produced at Fordham University, and is tracking toward an Off-Broadway production. This game brings the story to life as an interactive experience -- part visual novel, part exploration, built by the playwright herself.

## Features

- Branching narrative powered by Ink.js
- Visual novel-style presentation with PixiJS rendering
- Ambient audio and sound effects via Howler.js
- Multiple story paths based on player choices
- Character art and campsite environments

## Tech Stack

- Next.js 16 + TypeScript
- Ink.js (narrative scripting engine)
- PixiJS (2D rendering)
- Howler.js (audio)
- Tailwind CSS 4

## Quick Start

```bash
npm install
npm run dev
# Open http://localhost:3000
```
